# MEDIA-SERVER
## Description
ivekit 을 핸들링 하기위한 중계서버


## Installation

```bash
$ yarn install
```

## Docker-compose

```bash
$ docker-compose up
```

## Execution

```bash
# local
$ yarn start:dev

# alpha
$ yarn start:alpha

# beta
$ yarn start:beta
```

## Swagger uri

- https://alpha-api.maxverse.io/media/api
